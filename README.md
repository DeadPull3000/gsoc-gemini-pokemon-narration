# Gemini-Powered Ecosystem Narration (GSoC Entry Task)

This repository contains my entry task for the Google Summer of Code proposal. It demonstrates the ability to ingest ecosystem data (Pokémon-themed), engineer a prompt, and use the Google Gemini API to generate a concise, 2-4 sentence narrative summary.

## Repository Structure

- `data/` - Contains the sample dataset (`pokemon_events.json`).
- `docs/` - Contains project documentation, implementation plans, and database schema diagrams.
- `src/` - Contains the Python script for summarization.
- `outputs/` - Generated sample outputs.
